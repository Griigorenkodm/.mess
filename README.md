# Mess — локальный мессенджер (HTML/CSS/JS/TS)

Функции:
- переписка в чатах (включая групповые)
- переключение темы (светлая/тёмная)
- редактирование **своих** сообщений
- удаление **своих** сообщений “для всех”
- сохранение в `localStorage`
- синхронизация:
  - без сервера: между вкладками (через `BroadcastChannel`)
  - с сервером: между всеми клиентами (WebSocket)

## Запуск

1) Установите Node.js (если ещё нет).
2) В папке проекта:

```bash
npm install
```

### Вариант A: с сервером (рекомендуется)

```bash
npm run start:local
```

Откройте в браузере: `http://localhost:3000`

### Вариант B: без сервера (локально)

```bash
npm run build
```

Откройте `index.html` в браузере.

Подсказка: режим без сервера синхронизирует только вкладки в одном браузере.

## Разработка

```bash
npm run watch
```

После изменений обновляйте страницу.

## Деплой на Render (домен вместо localhost)

1) Загрузите проект в GitHub (Render подтягивает код из репозитория).
2) В Render создайте **New Web Service** и выберите репозиторий.
3) Render сам прочитает `render.yaml`, либо выставьте вручную:
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
4) После деплоя Render даст домен вида `https://<service>.onrender.com`.

Ссылки на чаты будут такого вида:
- `https://<service>.onrender.com/?chat=<id>`
